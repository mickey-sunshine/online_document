## For Developers

The documentation is based on [RST file format](https://www.sphinx-doc.org/en/master/usage/restructuredtext/basics.html) and compiled through the [Sphinx](https://www.sphinx-doc.org/en/master/).
As a contributor, you only need to update the RST source files.
The documentation is automatically built, updated and hosted on the [Continous Integration System](apps.readthedocs.com).

> [!NOTE]
> Version number follows AAEE releases. Please do not update yourself!

> Please do not include developer manuals in the user guide, which is for internal use only.
